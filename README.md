#  BUET C Programming Term Final Exam Solutions

A collection of solved C programming questions from Term Final Examinations across various departments of BUET.

### 🎯 Purpose

This repository provides tested, solved C code for practice. The questions are provided in a separate PDF file (e.g., `Questions.pdf`) that must be downloaded first. Try to solve by yourself, then match the answers.

### ✨ Coverage

The problems rigorously cover all core introductory C programming concepts:
* **Basics:** Data Types, Operators, Conditions (`if`/`switch`)
* **Flow Control:** Loops (`for`, `while`, `do-while`)
* **Data Structures:** Arrays (1D/2D), Strings
* **Advanced:** Functions, Pointers
